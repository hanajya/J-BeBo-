1.1.发送验证码

请求路径

POST /api/send-code

请求参数

参数名 位置 类型 必须 说明
mobile body string Y 手机号
type body int Y 验证码类型 1登陆 2注册 3找回密码
captcha body string N type=2时，必传 注册时的图形验证码

正确返回示例

{
    "status": 200,
    "message": "验证码已发送"
}

错误返回示例

{
    "status": 400,
    "message": "发送太过频繁，请稍后再试"
}

无
